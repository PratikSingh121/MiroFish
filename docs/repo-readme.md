# `README.md` — Repository Readme (Chinese)

## Overview

The root `README.md` is the main public-facing repository introduction in Chinese. It explains what MiroFish is, how the five-step workflow works, how to run it from source or Docker, and where to find demos and community links.

It is written for external readers rather than internal developers, so it blends product positioning, screenshots, quick-start instructions, and community information.

---

## Main Sections

### Hero and branding block

Provides:
- project logo
- tagline in Chinese and English
- badges for stars, forks, Docker, DeepWiki, and social links
- quick language switch to the English readme

### Project overview

Explains the core product idea: upload seed material, generate a high-fidelity digital social world, simulate agent behavior, and receive a prediction report plus an interactive simulated world.

### Vision

Frames MiroFish as both:
- a macro decision-rehearsal lab
- a micro creative sandbox for personal exploration and fiction-style experimentation

### Live demo and screenshots

Points readers to the hosted demo and embeds multiple UI screenshots.

### Demo videos

Includes video links for example scenarios such as Wuhan University public-opinion forecasting and *Dream of the Red Chamber* ending prediction.

### Workflow

Documents the five-step pipeline:
1. graph build
2. environment setup
3. simulation
4. report generation
5. deep interaction

### Quick start

Documents two deployment modes:
- source-code deployment
- Docker deployment

It also lists required environment variables such as:
- `LLM_API_KEY`
- `LLM_BASE_URL`
- `LLM_MODEL_NAME`
- `ZEP_API_KEY`

### Community and recruiting

Contains community contact information and a recruiting email.

### Acknowledgments

Credits Shanda Group and the OASIS/CAMEL-AI open-source stack.
